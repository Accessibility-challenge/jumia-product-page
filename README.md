A group project to identify and fix accessibility issues on Jumia Nigeria's product page.

##  Project Overview
We are fixing accessibility issues on Jumia's product page. 

## What We're Fixing

### Issue 1: Heading Structure
The page doesn't use proper heading tags for screen reader navigation.

### Issue 2: Keyboard Navigation
Users can't properly navigate the site using only the keyboard.

### Issue 3: Missing ARIA Labels
Icon buttons have no text descriptions for screen readers.

### Issue 4: Dynamic Content Not Announced
Shipping fee changes aren't announced to screen reader users.

## Timeline ~ One Week
- DAY1:  Audit and document issues
- DAY2-5: Apply fixes and test
- DAY3: Validation and documentation

## How We're Working
1. Each group member claims an issue
2. Test fixes with screen readers
3. Document what we changed
5. Submit fixes to this repo
